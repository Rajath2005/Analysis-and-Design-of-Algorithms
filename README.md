## 📖 About

**ADA Hub** is a comprehensive educational website for the **Analysis and Design of Algorithms** lab (VTU BCSL40A). It provides all 13 required programs written in C, each with a full learning package: algorithm explanation, step-by-step procedure, complete source code, sample I/O, dry-run trace, time & space complexity, real-world applications, advantages & disadvantages, and viva Q&A.

> Built to help fellow VTU Computer Science students ace their ADA lab exams and interviews.

## 🌐 Website Features

Each algorithm page on [ada-lab.vercel.app](https://ada-lab.vercel.app) includes:

- **📝 Introduction** — What the algorithm is and its historical context
- **🎯 Problem Statement** — Formal definition of the problem being solved
- **🌍 Real-World Applications** — Practical uses across industry
- **💡 Algorithm Explanation** — Detailed conceptual walkthrough
- **📋 Step-by-Step Procedure** — Numbered implementation guide
- **💻 Complete C Program** — Syntax-highlighted, copy-ready source code
- **🔢 Sample Input & Output** — Test cases to verify your program
- **🔍 Dry Run / Trace** — Manual execution trace through an example
- **⚡ Complexity Analysis** — Time and space complexity badges
- **✅ Advantages & Disadvantages** — Comparative analysis
- **❓ Viva Questions & Answers** — Exam-ready Q&A for lab viva

## 🚀 Run the C Programs

```bash
# Clone the repository
git clone https://github.com/Rajath2005/Analysis-and-Design-of-Algorithms.git
cd "Analysis-and-Design-of-Algorithms/ADA Programs"

# Compile any program (example: Kruskal's)
gcc Kruskal_Algorithm.c -o kruskal

# Run it
./kruskal
```

## 💻 Run the Website Locally

```bash
cd ada-hub
npm install
npm run dev       # → http://localhost:5000
```

```bash
npm run build     # Build for production → ada-hub/dist/
npm run preview   # Preview the production build
```
